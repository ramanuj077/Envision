Mock Login Flow Web Application

A simple 3-page mock login application built using HTML, CSS, and JavaScript to demonstrate basic authentication logic, conditional statements, and page navigation.

Features

Login validation using hardcoded credentials (admin / 1234)

Dashboard with personalized welcome message

Profile page with static user details

Logout functionality using localStorage

Clean, professional white UI design

Fully client-side (no backend)
